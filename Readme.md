# DBArkDesktop

基于Unity-DragonBones 开发的一款桌面宠物程序

## 现在它能做什么?

1. 使用龙骨运行库随机播放动画
2. 鼠标穿透
3. 桌宠现在能够在桌面自由移动(可能会有bug)
4. 按键触发人物动作


## 按键
1. 方向左键	向左移动
2. 方向右键	向右移动
3. 方向上键 向上移动
4. 方向下键 向下移动
3. Z	坐
4. P 	特殊（指玩牌）
5. S	睡觉
6. R	放松
7. Space 攻击
8. T 	说话
